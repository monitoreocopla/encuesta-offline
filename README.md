# Encuesta offline – versión completa

Archivos:

- index.html: estructura de pantallas y campos.
- styles.css: estilos base.
- script.js: lógica de navegación, validación, departamentos/escuelas y guardado local.
- departamentos.js: departamentos generados desde la hoja de escuelas.
- escuelas.js: listado completo de escuelas con CUE-Anexo, departamento y etiqueta.
- manifest.json: configuración PWA.
- sw.js: service worker mínimo (sin cache agresivo).
